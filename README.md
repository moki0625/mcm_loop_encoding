# mcm_loop_encoding

## Virtual Environment install
This code is recommended to be run on a virtual environment. To install the environment:
```PowerShell
python -m venv mcm_venv
```
`mcm_venv` can be replaced by any name you like.

The virtual environment is activated by
```
mcm_venv\Scripts\Activate.ps1
```
This command should be run in **PowerShell** under the git repository folder `your_base_folder\mcm_loop_encoding` each time before running the encoding code.

Install the requirements
```PowerShell
pip install -r requirements.txt
```

## Running the code
```PowerShell
python src\main.py
```
